# Pomodoro Timer 极简番茄专注钟
轻量无依赖网页专注计时器，原生HTML/CSS/JS实现，一键部署GitHub Pages，面向开发者、学生提升时间管理效率。

## ✨ 核心功能
1. 标准25分钟工作 / 5分钟休息周期，一键切换模式
2. 开始、暂停、重置完整计时控制
3. 深色/浅色主题无缝切换，适配夜间使用
4. 计时结束音效提醒
5. 专注完成次数统计，记录每日专注成果
6. 移动端自适应布局，细腻hover动效与柔和阴影
