# Pomodoro Timer 极简番茄专注钟

轻量无依赖网页专注计时器，原生 HTML/CSS/JS 实现，可直接打开使用，也适合一键部署到 GitHub Pages，面向开发者、学生提升时间管理效率。

## ✨ 核心功能

1. 标准 25 分钟工作 / 5 分钟休息周期，一键切换模式
2. 开始、暂停、重置完整计时控制
3. 深色/浅色主题切换，并自动记住偏好
4. 计时结束音效提醒
5. 专注完成次数统计，并在本地保存
6. 移动端自适应布局，包含进度环、hover 动效与柔和阴影

## 🚀 快速使用

1. 克隆仓库到本地

```bash
git clone https://github.com/knglml/pomodoro-timer.git
cd pomodoro-timer
```

2. 直接打开 `index.html`，或使用任意静态服务器运行：

```bash
python3 -m http.server 8000
```

3. 在浏览器访问：

```text
http://localhost:8000
```
